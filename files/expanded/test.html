Voici comment faire un tableau:

Pour commencer, sachez qu'il est nécessaire d'utiliser les balises:

  • <table> </table> : Déclaration d’un tableau
  • <caption> </caption> : Donner le titre du tableau
  • <tr> </tr> : La ligne du tableau
  • <td> </td> : La case du tableau
  • <th> </th> : La case d’entête (par défaut en gras) du tableau
  • <td colspan=n> </td> : La case occupe n colonne
  • <td rowspan=n> </td> : La colonne occupe n lignes
  • <thead> </thead> : L'entête du tableau
  • <tbody> </tbody> : Corps du tableau
  • <tfoot> </tfoot> : Pied du tableau

Introduction à Flexbox

La fonction Flexbox est très utiliseé dnas la disposition graphique d'éléments encastrés, par exemple disposer trois smiley côte à côte pour récuperer l'avis de l'utilisateur sur la page par exemple.

Pour utiliser Flexbox, on va principalement passer par le code CSS.

Ici, nous allons disposer trois carrés et je vais vous montrer comment flexbox peut se rendre utile.

La vidéo

Pour inserer une vidéo, rien de plus simple, entrez :
<video src="(l'accès de votre vidéo).(Le format) mp4" controls width="600" height="600">Mettez la description
Faite suivre cette balise par </video>

cela donne:


Vous pouvez controler la taille de la vidéo avec les mêmes éléments que pour changer la taille d'une image.
